经典电影静态网站

生成结果：
- 已解析影片数量：2000
- 已生成首页、分类总览页、12 个独立分类页、排行榜页、搜索页、影片详情页、站点说明页。
- 每个 HTML 页面都已插入百度统计代码，代码位于页面底部，不会显示成页面文字。
- 影片封面与 Hero 图片路径使用网站顶级目录下的 1.jpg 到 150.jpg。
- 如需显示封面，请把 1.jpg、2.jpg ... 150.jpg 放到本 ZIP 解压后的根目录。
- 播放器使用详情页中的 m3u8 地址，并在浏览器中通过原生 HLS 或 hls.js 初始化播放。

建议部署方式：
- 上传整个目录到静态空间、Nginx、Apache、宝塔面板或对象存储静态站点。
- 入口文件为 index.html。
